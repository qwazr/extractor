Compile and build
=================

## Requirements

- [Maven 3.0 or newer](http://maven.apache.org/)
- [Java JDK 7 or newer](http://openjdk.java.net/install/)
 
## Get the source code

```shell
git clone https://github.com/qwazr/qwazr-extractor.git
```
    
## Compile

```shell
mvn clean package
```

## Package

The binary is in the target directory.
